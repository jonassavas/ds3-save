# ds3-save

Dark Souls III was an amazing experience the first time around. However, the game does not support cloud saves on Steam, resulting in the loss of my original savefile a few years back. As the itch to play this game again grew into something that could no longer be ignored, I had to find a solution for saving and backing up my file. Therefore, this repository contains my solution to saving and restoring my savefile on both Windows & Linux.

### Instructions:

Execute it and follow the instructions:
```

```


#### Linux:
Make sure its executable:
```
chmod +x ds3-save.sh
```

Execute it and follow the instructions:
```
./ds3-save.sh
```
