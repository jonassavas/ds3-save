# ds3-save

Dark Souls III was an amazing experience the first time around. However, the game does not support cloud saves on Steam, which resulted in the loss of my original savefile a few years back. As the itch to play this game again grew into something that could no longer be ignored, I had to find a solution for this. Therefore, this repository contains scripts to save and restore my savefile on both Windows & Linux.

### Instructions:

#### Windows:
Execute it and follow the instructions:
```
.\ds3-save.ps1
```


#### Linux:
Make sure its executable:
```
chmod +x ds3-save.sh
```

Execute it and follow the instructions:
```
./ds3-save.sh
```
